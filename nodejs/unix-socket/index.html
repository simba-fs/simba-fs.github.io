刪除 socket file

因為每次的 listen 都會新增一個 socket file
相當於佔用一個 port
如果不刪除的話就相當於 port 被佔用了
會噴錯
所以每次程式結束的時候要把這個 socket file 刪掉
然後在 /tmp 下的檔案會在開機後自動刪掉
所以如果部屬後發什麼問題重開機就對了！
